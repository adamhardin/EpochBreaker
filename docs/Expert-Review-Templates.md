## Change Proposal Checklist

Complete before submitting a change for review.

- [ ] Clear player problem defined
- [ ] Success criteria stated
- [ ] Player flow documented
- [ ] Touch ergonomics considered
- [ ] Procedural generation impact evaluated
- [ ] Difficulty curve impact evaluated
- [ ] Scoring and incentives impact evaluated
- [ ] Weapon progression impact evaluated
- [ ] Prototype or build link available
- [ ] Playtest notes attached
- [ ] Risk analysis attached
- [ ] Rollback plan defined
- [ ] QA validation plan defined
