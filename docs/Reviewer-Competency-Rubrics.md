# Reviewer Competency Rubrics

Each competency is scored on a 100-point rubric. Pass thresholds are specified per competency.

## Critical 1: 2D Platformer Game Feel & Tuning

**Rubric**
- Responsiveness and control tightness: 30
- Consistency across edge cases: 20
- Forgiveness tuning (coyote time, buffering): 15
- Reference parity (benchmark feel match): 20
- Documentation quality: 15

**Pass Threshold**: >= 85/100, no category below 12/20-equivalent.
